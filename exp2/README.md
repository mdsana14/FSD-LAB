 # Experiment 2: User Login System

## Objective
To create a simple login system using Node.js.

## Description
This app checks username and password and returns response.

## Technologies Used
- Node.js
- Express.js

## How to Run
1. Run: node server.js
2. Open browser: http://localhost:3000
